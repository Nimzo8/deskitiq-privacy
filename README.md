# DeskitIQ Privacy Policy

This repository hosts the public privacy policy for **DeskitIQ**, a Windows desktop cleanup and organization app published by **Nimzo**.

DeskitIQ primarily works locally on the user's Windows device. It scans desktop items locally, builds a cleanup preview, and can create local backup manifests and cleanup history so changes can be undone or restored.

## Files

- `index.html` — the public privacy policy page
- `.nojekyll` — tells GitHub Pages to serve the files as-is
- `README.md` — this repository guide

## Current Policy Assumptions

The current DeskitIQ privacy policy is based on the following:

- DeskitIQ works primarily locally on the user's device
- desktop analysis happens locally
- backup manifests, cleanup history, and restore information are stored locally on the user's device
- DeskitIQ does **not** upload user desktop files to Nimzo servers
- DeskitIQ does **not** use analytics, telemetry, crash reporting, ads, or third-party tracking services
- DeskitIQ does **not** perform automatic external update checks in the current version
