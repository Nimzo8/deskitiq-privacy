# DeskitIQ Privacy Policy

This repository hosts the public privacy policy for **DeskitIQ**, a Windows desktop cleanup and organization app published by **Nimzo**.

DeskitIQ primarily works locally on the user's Windows device. It scans desktop items locally, builds a cleanup preview, and can create local backup manifests and cleanup history so changes can be undone or restored.

## Files

- `index.html` - the public privacy policy page
- `.nojekyll` - tells GitHub Pages to serve the files as-is
- `README.md` - this setup guide

## Before Publishing

Edit `index.html` and replace these placeholders:

- `[EFFECTIVE DATE]`
- `[CONTACT EMAIL]`
- `[GITHUB USERNAME]`
- `[YEAR]`

Also confirm whether DeskitIQ uses any crash logging, telemetry, analytics, external update checks, ads, or third-party services. The included policy assumes the current version does not send analytics, telemetry, crash reports, advertising identifiers, or automatic update-check data to Nimzo.

## Publish With GitHub Pages

1. Create a new public GitHub repository, for example `deskitiq-privacy`.
2. Upload these files to the root of the repository:
   - `index.html`
   - `README.md`
   - `.nojekyll`
3. In GitHub, open the repository.
4. Go to **Settings**.
5. Go to **Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Choose the `main` branch and the `/ (root)` folder.
8. Save.
9. Wait for GitHub Pages to finish publishing.

Your privacy policy URL will usually look like:

```text
https://[GITHUB USERNAME].github.io/deskitiq-privacy/
```

Use that URL in Microsoft Partner Center as the DeskitIQ privacy policy URL.
